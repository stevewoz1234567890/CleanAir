# CleanCloud-2

This is the repository for cleancloud.cleanair.com

# Getting Started

1. Installation process

   - Node >= 12
   - npm >= 6
   - npm install in root folder and cd into client, run npm install again
   - in root folder npm run devfull (It will run both front-end and back-end including redis server)

2. Software dependencies

- This requires backend redis running so you need to install it globally.
	Or you can simply install this.
  https://www.memurai.com/get-memurai
